# Boilerplate Xamarin Forms Template 

## Dependencies:

1. Xamarin forms 4.3+
2. [Prism](https://github.com/PrismLibrary/Prism) 7.2 +
3. FFimageloading for SVG 2.4+
4. Xamarin Essentials 1.3.1+


## Main modules:

1. Startup module
2. Account moudle
3. Home module
4. Common module
5. Settings module
